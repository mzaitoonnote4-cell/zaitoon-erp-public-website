# Homepage content

**Edit `homepage-content.js` to change all public marketing copy.**

No build step. Refresh the browser after saving.

| Key | What it controls |
|-----|------------------|
| `seo` | Page title, meta description, OG image |
| `sections` | `showHero`, `showModules`, `showPricing`, etc. |
| `sectionOrder` | Render order of homepage blocks |
| `hero` | Headline, CTAs, trust badges, mockup KPIs |
| `modules.items` | Module cards |
| `why.items` | Benefit cards |
| `industries.items` | Industry cards |
| `howItWorks.steps` | Workflow steps |
| `security.points` | Security & permissions bullets |
| `pilot` | Pilot program banner + CTA |
| `faq.items` | FAQ questions and answers |
| `whatsappCta` | Final WhatsApp CTA block |
| `pricing.plans` | Pricing tiers (USD only — Starter/Growth/Business Pilot + Enterprise) |
| `pricing.addons` | Additional pricing options |
| `pricing.note` | Pilot pricing disclaimer |
| `news.items` | News placeholders (`showNews: true` to display) |
| `media.logoDark` / `logoLight` | Official ZA ERP logo PNG (header/footer) — do not replace with CSS/SVG |
| `media.logoMonogram` | Cropped monogram PNG for brand section |
| `media` | Hero mockup, OG image, other assets |
| `footer` | Links, contact, social |
| `settings` | ERP login URL, WhatsApp number, `market` (primary country — Iraq first) |

TypeScript types: `src/content/public-homepage.ts`
